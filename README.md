# TouchPlay

**Your Android phone. A real controller. Any PC game.**

[![Download](https://img.shields.io/github/v/release/shantoshdurai/TouchPlay?color=00D4FF&label=Download)](https://github.com/shantoshdurai/TouchPlay/releases/latest)

</div>

---

Connect over Wi-Fi or USB. Works with any game that takes a gamepad. Up to 4 phones at once for local co-op.

Comes with layouts for **Standard Gamepad**, **Forza Horizon**, and **Spider-Man 2** — or build your own in the editor.

**PC setup:** run `TouchPlay-Setup.bat` once. It handles everything.  
**Phone:** install the APK from [Releases](https://github.com/shantoshdurai/TouchPlay/releases/latest).

> Windows 10/11 · Android 8.0+
