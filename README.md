# TouchPlay

**Your Android phone. A real controller. Any PC game.**

[![Download](https://img.shields.io/github/v/release/shantoshdurai/TouchPlay?color=00D4FF&label=Download)](https://github.com/shantoshdurai/TouchPlay/releases/latest)

</div>

---

Connect over Wi-Fi or USB. Works with any game that takes a gamepad. Up to 4 phones at once for local co-op.

**Six tools in one app:** Gamepad · Mouse & Keys (scroll / pinch-zoom / right-click) · Screen Mirror · File Transfer · Virtual Cam · Projector.

Comes with layouts for **Standard Gamepad**, **Forza Horizon**, and **Spider-Man 2** — or build your own in the editor.

**PC setup:** run `TouchPlay-Setup.bat` once. It handles everything. (Or just grab `TouchPlay-Server.exe` from Releases — no Python needed.)  
**Phone:** install the APK from [Releases](https://github.com/shantoshdurai/TouchPlay/releases/latest).

**What's what in this folder**

| Path | Purpose |
|---|---|
| `client/` | Flutter phone app |
| `server/` | Python PC server |
| `docs/` | Roadmap + release notes |
| `TouchPlay-Setup.bat` | One-time PC install (packages, driver, firewall) |
| `TouchPlay-Server.bat` | Starts the server (day-to-day use) |
| `build-exe.bat` | Builds the standalone `TouchPlay-Server.exe` |
| `HOW-TO-USE.txt` | End-user guide shipped in the release zip |

> Windows 10/11 · Android 8.0+
