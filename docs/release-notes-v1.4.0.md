# TouchPlay v1.4.0 — The PS5-style redesign

One design language, everywhere. The console-style home menu now flows through the whole app.

## New
- **Home menu**: cover-art tiles with a selection-following blue glow, white Launch pill, Xbox-letter face buttons, and a living backdrop (drifting dust + lamp light)
- **Every screen matches**: the same ambient backdrop and white pill buttons across Gamepad, Mouse & Keys, Screen Mirror, File Transfer, Virtual Cam and Projector
- **Portrait support**: home menu, Mouse & Keys, File Transfer, Virtual Cam and Projector all rotate now
- **Mouse & Keys**: live preview chip shows what you're typing, voice (mic) input works, keyboard state can't get stuck
- **Dedicated screens** for Mouse & Keys and Screen Mirror; the PC server has a proper window UI
- A quiet **"Get the PC server"** helper (open or copy the download link) whenever a feature needs the PC

## Fixed
- Layout editor: controls dragged to the right side no longer get trapped under the inspector panel
- No more black flash on menu tiles at first open
- Search is a compact, centered palette that stays above the keyboard
